# t-SNE后处理调整 - 参数说明文档

## 📋 概述

**核心原理**：
1. 先进行标准t-SNE降维
2. 在2D空间中调整各组到目标距离
3. 保留整体结构，只改变组内紧凑性

---

## 🔧 可调整参数详解

### 1️⃣ **TARGET_DISTANCES** - 目标平均距离 

### **作用**：控制每组内部点之间的平均距离（单位：t-SNE空间距离）

**如何调整**：

- **数值越小**：组内越紧凑，点之间越靠近
- **数值越大**：组内越分散，点之间越远
- **设为None**：保持原始t-SNE的位置，不调整

**调整建议**：
- 要让某组**最紧凑**：设为最小值（如3.0）
- 要让某组**很分散**：设为较大值（如15-20）
- 要实现**平滑收敛**：让1→2→3→4单调递减
  ```python
  例如：20.0 → 15.0 → 10.0 → 5.0 → 3.0
  ```

**典型设置**：
- **强收敛效果**：`{1: 20, 2: 12, 3: 7, 4: 3, 5: 5}`
- **中等收敛**：`{1: 16, 2: 10, 3: 6, 4: 3, 5: 5}` ✓ 推荐
- **弱收敛效果**：`{1: 12, 2: 9, 3: 6, 4: 4, 5: 5}`

**示例效果**：
```
设置：{1: 20, 2: 10, 3: 5, 4: 2, 5: 4}
结果：Iter 1很分散 → Iter 2中等 → Iter 3较紧 → Iter 4极紧 → Iter 5稍松
```

---

### 2️⃣ **N_BACKGROUND** - 背景点数量

```python
N_BACKGROUND = 500
```

**作用**：控制灰色背景点的数量

**如何调整**：
- **300-500**：标准配置，足够平滑 ✓ 推荐
- **100-300**：较少，计算快但可能不够均匀
- **500-1000**：很平滑，但计算较慢
- **1000+**：不推荐，计算很慢，效果提升不明显

**建议**：保持500即可

---

### 3️⃣ **BACKGROUND_COMPACTNESS** - 背景椭圆紧凑度

```python                
BACKGROUND_COMPACTNESS = 0.25
```

**作用**：控制背景点形成椭圆的紧凑程度

**如何调整**：
- **0.15-0.20**：非常紧凑，椭圆很小
- **0.25-0.30**：中等紧凑 ✓ 推荐
- **0.35-0.50**：较分散，椭圆较大
- **0.50+**：非常分散，接近均匀分布

**效果对比**：
```
0.20: 紧凑椭圆，背景点聚集在中心
0.25: 适中，视觉效果好 ✓
0.40: 分散，背景点分布广
```

**建议**：0.25是经验最佳值

---

### 4️⃣ **TSNE_PARAMS** - t-SNE算法参数

```python
TSNE_PARAMS = {
    'perplexity': 40,
    'learning_rate': 150,
    'max_iter': 1500,
    'early_exaggeration': 12,
    'random_state': 42,
}
```

#### **perplexity** - 困惑度
**作用**：平衡局部和全局结构

**如何调整**：
- **5-15**：强调局部结构，聚类明显
- **30-50**：平衡局部和全局 ✓ 推荐
- **50+**：强调全局结构

**建议**：
- 样本少（<100）：用20-30
- 样本中等（100-500）：用30-50 ✓
- 样本多（>500）：用50-100

#### **learning_rate** - 学习率
**作用**：控制优化速度

**如何调整**：
- **50-100**：慢，更稳定
- **100-200**：适中 ✓ 推荐
- **200-1000**：快，可能不稳定

**建议**：150是经验值，一般不需要改

#### **max_iter** - 最大迭代次数
**作用**：t-SNE优化的迭代次数

**如何调整**：
- **500-1000**：快速预览
- **1000-2000**：标准设置 ✓ 推荐
- **2000+**：精细调优，计算慢

**建议**：1500足够，除非结果不收敛才增加

#### **early_exaggeration** - 早期夸张
**作用**：早期阶段夸大簇间距离，让聚类更明显

**如何调整**：
- **4-8**：弱夸张
- **12-16**：标准夸张 ✓ 推荐
- **20+**：强夸张，聚类非常明显

**建议**：12效果好，可以试试15让聚类更明显

#### **random_state** - 随机种子
**作用**：保证结果可重复

**建议**：**不要改**，除非想要不同的随机初始化

### 5️⃣ **ITER_COLORS** - 迭代组配色

```python
ITER_COLORS = ['#FF6B6B', '#4ECDC4', '#45B7D1', '#FFA07A', '#98D8C8']
```

**作用**：定义Iter 1-5的颜色（十六进制颜色码）

**预设配色方案**：

**方案1 - 当前配色**（混合色系）✓ 默认
```python

# 红色    青色      蓝色      橙色      绿色
```

**方案2 - 暖色系**
```python
['#FF6B6B', '#FF8E53', '#FFA94D', '#FFD93D', '#6BCF7F']
# 红→橙→黄→绿渐变
```

**方案3 - 冷色系**
```python
['#667EEA', '#64B6F7', '#48B0F7', '#3EC1D3', '#9CECFB']
# 紫→蓝渐变
```

**方案4 - 彩虹**
```python
['#E74C3C', '#F39C12', '#F1C40F', '#2ECC71', '#3498DB']
```

### 6️⃣ **ITER_MARKERS** - 迭代组标记形状

```python
ITER_MARKERS = ['o', 's', '^', 'D', 'v']
```

**作用**：定义Iter 1-5的标记形状

**可用形状**：
```
'o'  圆形      's'  方形
'^'  上三角    'v'  下三角
'<'  左三角    '>'  右三角
'D'  菱形      'p'  五边形
'*'  星形      'h'  六边形
'+'  加号      'x'  叉号
```

**预设方案**：

**方案1 - 当前配置**（几何形状）✓ 默认
```python
['o', 's', '^', 'D', 'v']
# 圆 方 上三角 菱形 下三角
```

**方案2 - 全三角系**
```python
['o', '^', '>', 'v', '<']
# 圆 上 右 下 左
```

**方案3 - 混合风格**
```python
['*', 'D', 'p', 'h', 's']
# 星 菱形 五边形 六边形 方
```

**建议**：保持默认即可，区分度高

### 7️⃣ **OUTPUT_DIR & OUTPUT_PREFIX** - 输出设置

```python
OUTPUT_DIR = '/mnt/user-data/outputs'
OUTPUT_PREFIX = 'tsne_final'
```

**作用**：
- `OUTPUT_DIR`：输出文件夹路径
- `OUTPUT_PREFIX`：输出文件名前缀

**生成的文件**：
```
{OUTPUT_PREFIX}_detail.png       # 详细版图
{OUTPUT_PREFIX}_simple.png       # 简化版图
{OUTPUT_PREFIX}_coordinates.xlsx # 坐标数据
```

**如何修改**：
```python
OUTPUT_PREFIX = 'my_tsne_20260123'
# 生成：my_tsne_20260123_detail.png 等
```

## 🎯 常见调整场景

### 场景1：让Iter 4更紧凑
```python
TARGET_DISTANCES = {
    0: None,
    1: 18.0,
    2: 12.0,
    3:  7.0,
    4:  1.5,  # ← 改小！
    5:  4.0,
}
```

### 场景2：让所有迭代更分散
```python
TARGET_DISTANCES = {
    0: None,
    1: 25.0,  # ← 全部放大
    2: 18.0,
    3: 12.0,
    4:  8.0,
    5: 10.0,
}
```

### 场景3：平滑的线性收敛
```python
TARGET_DISTANCES = {
    0: None,
    1: 20.0,
    2: 16.0,  # 每次减4
    3: 12.0,
    4:  8.0,
    5: 10.0,
}
```

### 场景4：更明显的背景椭圆
```python
N_BACKGROUND = 800  # 增加点数
BACKGROUND_COMPACTNESS = 0.20  # 更紧凑
```

### 场景5：更强的聚类效果
```python
TSNE_PARAMS = {
    'perplexity': 30,  # 减小
    'learning_rate': 150,
    'max_iter': 2000,  # 增加
    'early_exaggeration': 16,  # 增大
    'random_state': 42,
}
```
